PRODUCT OVERVIEW
Product Name: ShiftMate
Purpose: A web and mobile app that allows users to easily log and manage their work shifts.
It includes calendar visualization, recurring shifts, smart dashboards, exportable reports, and push notifications.

Target Audience:

Support workers

ABN/Freelance contractors

Shift-based professionals

üöÄ CORE FEATURES (MVP)
Shift logging by date, start and end time (15-min intervals)

Recurring shifts: daily, weekly, custom patterns

Calendar view: monthly layout with color-coded shifts

Dashboard with:

Total hours for current and previous week

Daily averages

Days without entries

Smart notification system:

Push alert if no shift is logged for 3+ days

Exportable reports:

PDF and CSV

Filters by custom date ranges

Totals by day, shift, and range

User-based data isolation:

Each user accesses only their data

Authentication via Email or Google

Frontend:

Backend:

Node.js + Express (or Next.js API Routes)

Service/repository/controller pattern

Database:

Supabase with PostgreSQL and built-in Auth

Structured, normalized schema

Frontend: Zod/Yup

Backend: Joi or middleware validators
